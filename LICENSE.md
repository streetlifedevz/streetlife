## 🔧 Troubleshooting & Common Issues

### ❌ GUI Not Appearing After Execute

**Cause:** The script executed but the GUI rendered off-screen or failed to initialize.

**Fix:**
1. Press the executor's Re-Execute button once
2. If still absent, type `/resetcharacter` in chat and re-execute
3. Verify your executor supports `Drawing` and `GuiService` API calls

---

### ❌ Silent Aim Not Hitting Targets

**Cause:** Server-side lag or incorrect bone target configuration.

**Fix:**
1. Change `AimbotBone` from `"Head"` to `"Torso"` in config
2. Reduce `AimbotFOV` to `80–100` for tighter lock
3. Ensure no other aim module is running simultaneously

---

### ❌ Infinite Money Not Working

**Cause:** Game update changed the farm trigger event names.

**Fix:**
1. Check the **Changelog** for the latest patch notes
2. Download the newest version from the link above
3. Clear your executor's cache folder and re-inject

---

### ❌ Vehicle Spawner Crashes Server

**Cause:** Spawning too many vehicles in rapid succession triggers server garbage collection.

**Fix:**
1. Add a 3–5 second delay between spawns
2. Despawn current vehicles before spawning new ones using the GUI's **Clear Spawned** button

---

### ❌ Script Not Loading on Mobile

**Cause:** Executor lacks proper injection timing on mobile Roblox builds.

**Fix:**
1. Wait until the game fully loads (past the loading screen) before injecting
2. Enable **Delayed Injection** mode in your mobile executor settings
3. Restart both Roblox and the executor, then try again

---

### ❌ Anti-AFK Failing (Getting Kicked)

**Cause:** The randomized input simulation isn't recognized by the server's AFK detection.

**Fix:**
1. Increase movement simulation frequency in config
2. Enable the **Auto-Rejoin** module as a safety fallback
